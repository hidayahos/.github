<div align="center">

| Layer | Name | Description |
|:-----:|------|-------------|
| **1** | ☁️ CloudBrowsing DNS | Blocks 99%+ harmful domains at DNS level |
| **2** | 🚫 eBlocker Islamic Blocklist | 25M+ entries blocking ads & haram content |
| **3** | 🔒 DNS Hijack Prevention | Forces all DNS through filtered resolver |
| **4** | 🦊 Firefox AppArmor Profile | 30-day strict browser confinement |
| **5** | 📦 Firejail Browser Sandbox | Filesystem-level browser isolation |
